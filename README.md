# Fluxstore Web

Sitio estático responsive para una tienda de vapes, optimizado para celular.

## Cómo editar

- Cambiar número de WhatsApp: abrir `script.js` y editar `WHATSAPP_NUMBER`.
- Cambiar productos, precios o sabores: editar el array `products` en `script.js`.
- Cambiar imágenes: reemplazar archivos dentro de `assets/` y actualizar la ruta en `script.js`.

## Cómo publicar rápido

Podés subir la carpeta a Netlify, Vercel o GitHub Pages. También funciona abriendo `index.html` en el navegador.

## Importante

Incluye gate +18 y aviso legal. Revisá las normas locales antes de vender o promocionar productos de vapeo.
